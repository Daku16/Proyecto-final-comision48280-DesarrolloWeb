Skilset

HTML

HTML (HyperText Markup Language) is a markup language used to create web pages. HTML defines the structure and content of a web page, allowing developers to create titles, paragraphs, lists, links, images, and other multimedia elements to display in a web browser. HTML is a universal standard and is compatible with all web browsers, allowing developers to create accessible and functional content on all devices.

CSS

CSS (Cascading Style Sheets) is a style language used to describe the visual presentation of a web page written in HTML. CSS allows developers to control the appearance of a web page, including colors, fonts, margins, sizes, and positions of elements on the page. With CSS, developers can separate the content and presentation of a web page, making maintenance easier and enabling responsive design for different devices and screens.

Boostrap

Bootstrap is an open-source design framework used to develop modern and uniform looking websites and web applications. Bootstrap provides a set of pre-built components and CSS styles that can be used to create a wide range of web designs. These components include navigation menus, forms, tables, buttons, modals, and many others. Bootstrap is also mobile-friendly and provides responsive design that automatically adjusts to different screen sizes.

Sass

Sass is a CSS preprocessor that provides a more advanced and powerful syntax allowing developers to write CSS more efficiently and maintainably. Sass features such as variables, mixins, nesting and inheritance are not available in traditional CSS. With Sass, developers can create more complex and sophisticated stylesheets with less code, saving time and improving code quality.
